# Compliance Folder README

This folder contains all compliance documentation for Clepto.io, organized by year.

---

## Folder Structure

```
/compliance/
├── 2025/                           # Current year
│   ├── README.md                   # This file (index)
│   ├── vendor-assessments/         # Third-party vendor reviews
│   ├── pentest-reports/            # Penetration test results
│   ├── audit-findings/             # Security audit results
│   ├── training-certificates/      # Security training completion
│   └── risk-assessments/           # Client/project risk assessments
│
├── HIGH-RISK-AI/                   # EU AI Act conformity docs
│   └── workflow-name/              # Per-workflow documentation
│       ├── conformity-assessment.md
│       ├── risk-assessment.md
│       └── human-oversight.md
│
└── templates/                      # Reusable templates
    ├── vendor-assessment-template.md
    ├── risk-assessment-template.md
    └── client-security-briefing.pptx
```

## Required Documents by Standard

### SOC 2 Type II
- [ ] Security policies
- [ ] Access control logs
- [ ] Change management records
- [ ] Incident response records
- [ ] Vendor assessments
- [ ] Risk assessments
- [ ] Penetration test reports

### ISO 27001
- [ ] ISMS policy
- [ ] Risk assessment methodology
- [ ] Statement of Applicability
- [ ] Asset inventory
- [ ] Business continuity plan

### GDPR
- [ ] Privacy policy
- [ ] Data processing agreements
- [ ] Records of processing activities
- [ ] Data breach notification procedures
- [ ] Data subject request procedures

### EU AI Act
- [ ] High-risk AI classification
- [ ] Conformity assessments
- [ ] Human oversight documentation
- [ ] Post-market monitoring plans
