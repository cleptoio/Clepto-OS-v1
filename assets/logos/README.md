# PLACEHOLDER - Clepto Logo

This directory should contain:

- `clepto-light.svg` - Light version of Clepto logo (for dark backgrounds)
- `clepto-dark.svg` - Dark version of Clepto logo (for light backgrounds)
- `favicon.ico` - Favicon for browser tabs
- `clepto-icon.png` - Square icon for various uses

## How to Add Your Logo

1. Place your logo files in this directory
2. Update references in:
   - Environment variables (`.env`)
   - Frontend code (`apps/clepto-crm/packages/twenty-front/`)
   - Email templates (`apps/clepto-mail/`)

## Logo Specifications

**SVG Logos**:
- Format: SVG (vector)
- Maximum width: 200px
- Transparent background
- Color scheme: 
  - Light: Use #0bd7d4 (cyan) for branding elements
  - Dark: Use #0e172f (navy) for branding elements

**Favicon**:
- Format: ICO or PNG
- Size: 32x32 pixels or 64x64 pixels
- Simple, recognizable icon

**Icon**:
- Format: PNG
- Size: 512x512 pixels (will be scaled down as needed)
- Transparent background
- High resolution for quality
